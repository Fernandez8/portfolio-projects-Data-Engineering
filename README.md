# Project 1 – Data Architecture

## Architecture Overview

Ce projet présente une architecture de données moderne conçue pour collecter, transformer, stocker et analyser des données à grande échelle.  
L'objectif est d'assurer la scalabilité, la fiabilité et la performance dans des environnements de production.

### 🔍 Schéma de l'architecture

![Architecture Diagram](project-1-data-architecture/datenarchitekt.png)

---

## Composants principaux

- **Kafka / REST API** → ingestion des données en temps réel ou batch  
- **Spark / Airflow** → traitement des données  
- **Data Lake (S3 / HDFS)** → stockage brut  
- **PostgreSQL / Snowflake** → stockage structuré  
- **Power BI / Metabase** → visualisation

---

## 💡 Objectifs pédagogiques

- Concevoir une architecture data robuste
- Automatiser des flux de données
- Travailler avec des outils big data utilisés en entreprise
